	This software books movie tickets.To use this software,you need account created in software.
By creating account you will get account ID,which you have to use to login.
	To run this software you have type "make" command in terminal,then type "./project".
	This program uses three files user.txt, exe.txt, tic.txt.
		user.txt--To store user info.
		exe.txt-- To store data entered executive.
		tic.txt-- To save booked ticket info.
	This software uses following functions:
		user booker--To book tickets.
		history--To view booked ticket history.
		ulogin--For user login.
		create--To create new account.
		elogin--For executive login.
		release--To wiew rcecnt released movies.
		home--To print menu.
